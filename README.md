# FumazzoAdmin

Administration of the Fumazzo app. 

# Status

*In development*.

Read the CHANGELOG.MD at the root of the project.

# Description

This app has the objective of administrating entries of the Fumazzo app: adding, editing and removing items, like:
- Meals,
- Products,
- Stock entries,
- Categories,
- Workers,
- Etc...

# Requirements

To start the app you must have:

1 - Python 3.11+.

2 - Install the requirements.txt at the root of the project. (It is advised to create and connect to a virtual environment(venv) before installing the requirements.txt.):
    - 'python -m venv venv'
    - 'venv\Scripts\activate' #Windows
    - 'source venv/bin/activate' #Linux/Mac
    - 'pip install -r requirements.txt' #Make sure that you are at the root of the project

3 - The .env file is also required to connect to the database and others sensitive informations.
